Guess the Number

You are going to write a program that asks a user to guess a number between 1 and 100. It will tell the user if there guess was larger or smaller than the secret number, and let them keep guessing until they guess the number.

Before starting the problem, here are some key Python concepts you will need to write this App.

Key Concepts

Variables

Variables name and hold values. If you assign a value—a number, a string, etc—to a variable you can use the variable name to access that value later. Variables are created and updated with the = character.


            foo = "Hello, coder!"
            print(foo)

Lists

Lists contain multiple values. Lists will tell you their size and what their values are. If you try to access a value beyond the size of the List you will get an error.

Usage

  • Create a list by putting the values between [ and ] and separating them by commas. 
    
              foo = ["flowers", 3, False]
  • Access lists values by putting the index number in brackets after the List variable name. 
    
              foo[1]
  • Add an item to a list with append. 
    
              foo.append(4)
              foo # So the REPL displays foo.
Examples

          # Define Lists
          x = [1, 2, 3, 4, 5]
          y = ["elliot", "oliver", "lola"]

          # Size of arrays
          len(x)
          len(y)

          # Values in arrays
          x[0]
          y[0]
          x[3]
          y[3]

          x[len(x) -1]

Random Numbers

In order to give different answers you will need something random. Python provides a number of options in its random library. First you must import the function you need—in this case randint. randint is called with randint(a, b) and returns a number that is equal to or greater than a and eqaul to or less than b.


        from random import randint

        # Generate numbers
        randrange(0,10)
        randrange(0,10)
        randrange(0,10)

        # Random range based on array size.
        randrange(0, len(x))
        randrange(0, len(x) -1)

Conditionals

You will need conditional code to tell the user if their answer was correct, and to determine when to exit the loop. An if statement executes code if the exprtession it is given is true. An if can be followed by elif and else. There can be multiple elifs but else must be the final clause. 


        something = False
        somethingElse = False
        anotherElse = True

        if (something):
          print("something")
        elif (somethingElse):
          print("somethingElse")
        elif (anotherElse):
          print("anotherElse")
        else:
          print("else")

The something in the if statement is either True or False. To compare numbers and produce a boolean (True or False) value, you use comparison operators.

==
Equal; returns True if the two numbers are the same.
>
Greater than; returns True if the first number is greater than the second.
<
Less than; returns True if the first number is less than the second.
>=
Greater than or equal; returns True if the first number is greater than or equal to the second.
<=
Less than or equal; returns True if the first number is less than or equal to the second.

Loops

Use a loop to repeat code more than once. There are various loop forms, and they can check to see if they should continue. For this, use a loop that always continues and use break when you want to exit.


          while(true):
              # Do some stuff
              if (done):

Guess the Number

You are going to write a program that asks a user to guess a number between 1 and 100. It will tell the user if there guess was larger or smaller than the secret number, and let them keep guessing until they guess the number.

Start out by setting picking the secret number, then prompt the user and give them feedback until they get the correct answer.

This is very much like the logic used in your favorite video games!

Suggested Steps

  • Generate the random answer (number between 1 and 100?)
  • Communicate to the user what they are guessing (number between 1 and 100?)
  • Ask the user for a guess.
  • Check the guess against the answer.
  • Communicate whether the answer was correct, too small, or too large.
  • Finish if correct.
  • Go back to asking for a guess if incorrect.

Challenges

  • Let the user specify the range (the smallest and largest numbers) the correct answer can be. What will you do if the user gives you a range where the first number is smaller than the second? What will you do it the user does not give you a number?
  • Tell the user how many guesses it took them to get the answer.
  • Keep track of all the users guesses and show them when the user gets the correct answer. You could even make fun of really bad guesses, and celebrate when users don't need many guesses!
